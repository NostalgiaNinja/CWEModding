Chao World Extended Modding Documentation

Creating custom hats and fruit for Chao Gardens with Chao World Extended can be a long and complicated process, but this document intends to ease users into understanding the core concepts of how to mod using CWE's modding support functions. While the documentation does go through all the groundwork to get you started, there will be some pre-requisites expected of you before going ahead. Each section of this guide will provide a small subsection explaining the pre-requisites of what it will go through before starting the mod.
